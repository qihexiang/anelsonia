    This overload is designed for internal using, two parameters are the next function and the first function. If you'd like to compose just 2 functions it's no problem using this, otherwise it's recomended using first overload of this function.

    This function has 3 generic parameters, P is the parameter of lastFn, T is the return type of lastFn and the parameter type of nextFn, and R is the return type of nextFn.
